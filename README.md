# 🚀 Mini Social Media Platform with AI Chatbot

A Django-based social media web application that allows users to create posts, upload images and videos, like, comment, follow other users, search users, manage profiles, and interact with an AI chatbot.

---

## 📖 Features

* 🔐 User Registration & Login
* 👤 User Profile
* 🖼️ Upload Profile Picture
* ✏️ Edit Profile
* 📝 Create Posts
* 📷 Upload Images
* 🎥 Upload Videos
* ❤️ Like / Unlike Posts
* 💬 Comment on Posts
* 👥 Follow / Unfollow Users
* 🔍 Search Users
* 🗑️ Delete Own Posts
* 🤖 AI Chatbot Assistant
* 🛠️ Django Admin Panel
* 📱 Responsive User Interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Putta-bhavani/CodeAlpha_socialMedia.git
```

### Move to the Project Folder

```bash
cd CodeAlpha_socialMedia
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

If you don't have a `requirements.txt` file:

```bash
pip install django pillow
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Run the Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
